subscriptions.apps module¶

Subscriptions app configuration.

This app manages user subscriptions to publishers and journalists. It defines the AppConfig class for Django to recognize this app and sets default behaviors such as the primary key field type.

class subscriptions.models.Subscription(*args, **kwargs)¶

Bases: Model

Represents a reader’s subscription to a publisher or journalist.

Attributes:

user (ForeignKey): The reader who subscribes. publisher (ForeignKey, optional): The publisher the user subscribes to. journalist (ForeignKey, optional): The journalist the user subscribes to. created_at (DateTimeField): Timestamp when the subscription was created.

Notes:

Each subscription must be unique for a combination of user, publisher, and journalist.
